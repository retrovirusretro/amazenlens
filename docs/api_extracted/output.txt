Neden API Satmak SaaS'tan Daha Büyük Olabilir?
AmazenLens'in şu an bir SaaS platformu olarak çalışıyor. Kullanıcılar aylık plan alıyor, arayüzü kullanıyor ve çıkıyor. Bu model işe yarıyor — ama gelir tavanı var.
API monetizasyonu farklı bir oyun. Burada müşteriniz artık bireysel satıcı değil — Türkiye'deki bir e-ticaret ajansı, Almanya'daki bir SaaS startup'ı veya Japonya'da bir toptan satış platformu. Tek bir B2B API müşterisi, yüzlerce bireysel SaaS müşterisinin getirisini aşabilir.
Stripe, Twilio ve SendGrid'in ortak noktası nedir? Hepsi bir servisi önce kendi ürünlerine entegre etti, sonra o servisi API olarak sattı. AmazenLens'in algoritmaları aynı potansiyele sahip.
Rakamlarla somutlaştıralım: AmazenLens'in Niş Skoru algoritması bir e-ticaret ajansının 200 müşterisine hizmet vermek için API aracılığıyla sorgu yapabilir. Bu ajans aylık $300 öder. 10 böyle ajans: $3.000/ay — bu rakama SaaS tarafında ulaşmak çok daha zordur.
1. Satılabilecek API'ler: Envanter
AmazenLens'in şu an sahip olduğu algoritmalar ve veriler doğrudan API'ye dönüştürülebilir. Her biri için pazar potansiyeli, hedef müşteri profili ve fiyatlandırma yaklaşımı aşağıda incelenmiştir.

Kimlik Doğrulama
Tüm API istekleri bir API anahtarı gerektirir. Anahtarı developers.amazenlens.com adresinden alabilirsiniz. Her istek için Authorization header'ında Bearer token olarak gönderin.
# Her istekte bu header gereklidir:
Authorization: Bearer al_live_xxxxxxxxxxxxxxxxxxxxxxxx
# Python örneği:
import httpx
headers = {
"Authorization": "Bearer al_live_xxxxxxxxxxxxxxxxxxxxxxxx",
"Content-Type": "application/json"
}
# cURL örneği:

https://api.amazenlens.com/v1/niche/score
API Key Güvenliği
API anahtarınızı kaynak kodunuza gömmeyin. .env dosyasında AMAZENLENS_API_KEY değişkeni olarak saklayın. Her ortam (dev/staging/prod) için ayrı anahtar kullanın. Sızan anahtarı hemen revoke edin.
Rate Limiting
Rate limit bilgisi her yanıtın headerlarında yer alır. Limiti aşmanız durumunda 429 Too Many Requests döner — Retry-After headerını kontrol edin.
# Yanıt headerları:
X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 847
X-RateLimit-Reset: 1709856000
X-Usage-Month: 12450
X-Usage-Month-Limit: 25000
